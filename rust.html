Rust 文档标题 Rust

NAME
Rust - Rust是Mozilla Research开发的一种编程语言。 
1.Rust将对性能的低级控制与高级便利性和安全性保证结合在一起。
2.它无需垃圾收集器或运行时即可实现这些目标，从而可以将Rust库用作C的“替代品”。


Rust 发展之路
1.Rust的第一个版本0.1于2012年1月发布，经过3年的发展，发展如此之快，以至于直到最近为止，人们都不鼓励使用稳定的版本，相反，一般建议是使用每夜构建。

2.2015年5月15日，Rust 1.0发布，完全保证了向后兼容性。夜间构建中当前可提供对编译时间和编译器其他方面的改进。Rust已采用基于火车的发布模型，每六周定期发布。在Rust 1.0发行的同时提供了Rust 1.1 beta。

3.尽管Rust是一种相对较低级别的语言，但它具有一些功能概念，这些概念通常可以在高级语言中找到。这使得Rust不仅速度快，而且可以轻松高效地进行编码。
